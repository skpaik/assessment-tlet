# tlet

## Test project for Backend(fastAPI) and Frontend (React)

### How to run

Step 1:
- ```cd backend```
- ```python3 -m venv .venv```
- ```source .venv/bin/activate```
- ```pip install -r requirements.txt```
- ```bash run.sh```

Step 2
- ```cd frontend```
- ```npm install```
- ```bash run.sh```
